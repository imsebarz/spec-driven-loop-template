# UI Design

## Product feel

Simple, calm, kitchen-friendly. Clean cards, large tap targets, low cognitive load.

## Primary screens

- Weekly planner
- Recipe library
- Grocery list
- Pantry

## Layout

### Desktop
- Left nav
- Main planning surface
- Optional details panel

### Mobile
- Bottom nav
- Single-column screens
- Sticky primary actions

## Visual language

- soft neutrals
- one primary accent color
- strong emphasis on clarity over decoration
